tasks.txt储存了100张瓦片卫星影像的左上角和右下角经纬度坐标以及保存路径，用于指定需要爬取的天地图瓦片卫星影像。

Tianditu_serial.py是用于串行爬取天地图瓦片卫星影像的Python代码。

Tianditu_parallel.py是用于并行爬取天地图瓦片卫星影像的Python代码。

downloadUSGSdata.py是用于爬取USGS数据的Python代码。

showtif.py是用于可视化tif格式文件的Python代码。

USGS_NDVI_computation.py是用于依据红光波段和近红外波段图像计算NDVI并可视化的Python代码。

Edge_detecting_serial.py是用于串行边缘检测的Python代码。

Edge_detecting_threading.py是用于多线程边缘检测的Python代码。

Edge_detecting_multprocessing.py是用于多进程边缘检测的Python代码。

Edge_detecting.c是用于串/并行边缘检测的OpenMP代码。

Gaussian_filtering_serial.py是用于串行高斯滤波的Python代码。

Gaussian_filtering_threading.py是用于多线程高斯滤波的Python代码。

Gaussian_filtering_multprocessing.py是用于多进程高斯滤波的Python代码。

Gaussian_filtering.c是用于串/并行高斯滤波的OpenMP代码。

Contrast_enhancement_serial.py是用于串行对比度增强的Python代码。

Contrast_enhancement_threading.py是用于多线程对比度增强的Python代码。

Contrast_enhancement_multprocessing.py是用于多进程对比度增强的Python代码。

Contrast_enhancement.c是用于串/并行对比度增强的OpenMP代码。
